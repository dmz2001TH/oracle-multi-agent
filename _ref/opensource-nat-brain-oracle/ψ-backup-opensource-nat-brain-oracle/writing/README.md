# ψ-drafts/

> **Tier**: 1 (Ephemeral)
> **Git**: Ignored
> **Lifecycle**: Hours to Days

## Purpose

Output drafts before finalization. Blog posts, slides, proposals waiting to be exported.

## Rules

- **Work in progress only** - Not final outputs
- **Export when ready** - Move to external destination
- **Delete after export** - Don't accumulate

## Structure

```
ψ-drafts/
├── blog-*.md          # Blog post drafts
├── pitch-*.md         # Presentation drafts
├── proposal-*.md      # Proposal drafts
└── *.md               # Other drafts
```

## Lifecycle

```
ψ-context/ --> ψ-drafts/ --> External (blog, slides, etc.)
(research)     (draft)       (published)
```
