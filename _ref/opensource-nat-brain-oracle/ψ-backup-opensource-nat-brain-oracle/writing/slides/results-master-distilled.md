## Universal Style Guide

### Color Palette (strict, all series)
- **Base**: Monochrome ink-wash (charcoal, blacks, whites, grays) — 70-80% of image
- **Energy/Flow**: Cyan (#00FFFF) — 15-20% — movement, connection, warnings, danger signals
- **Completion/Decision**: Gold (#FFD700) — 5-10% — endpoints, choices, truth, consequences
- **No other colors**

### Typography (Thai-compatible, all free)
- **Noto Sans Thai**: Headers, titles (modern, clear)
- **Prompt**: Body text, concept labels (modern, clean)
- **Kanit**: Philosophical statements, core principles (geometric, serif)
- **Sarabun**: Supporting text, poetic elements (readable, flowing)
- **English**: Noto Sans (consistent international look)

### Technical Specifications
- Aspect ratio: 16:9 (landscape)
- Quality: --q 2 (maximum detail)
- Output: PNG (lossless)
- Resolution: 1920x1080px minimum, 3840x2160 preferred
- File naming: `[NUMBER]-[slug].png`

### Composition Principles
- Asymmetrical balance reflecting human uncertainty
- Human figures at center in most prompts (agency perspective)
- Negative space as meaning (gaps, voids, unknown)
- Thai text integrated as visual element, not overlay
- Avoid corporate aesthetic; embrace hand-drawn quality

## Image Generation Template

### Standard Preamble (copy before each prompt)

```
Create a monochrome ink-wash illustration in 16:9 aspect ratio (landscape).

Style requirements:
- Base: Monochrome ink-wash effect (hand-drawn, traditional aesthetic)
- Energy/Flow elements: Cyan color (#00FFFF or similar bright cyan)
- Completion/Decision elements: Gold color (#FFD700 or warm gold)
- No other colors except monochrome, cyan, and gold

Typography:
- Thai text: Use Noto Sans Thai font for authenticity
- Secondary Thai fonts acceptable: Prompt, Kanit, or Sarabun
- English text: Use Noto Sans
- All text rendered as design elements, not overlays

Format:
- Output: PNG (high quality, lossless)
- Aspect Ratio: 16:9 (1920x1080 minimum, 3840x2160 preferred)
- No branding, watermarks, or signatures

Visual principles:
- Avoid corporate aesthetic
- Embrace imperfection and hand-drawn quality
- Use fractals and geometric patterns to suggest self-similarity
- Thai text should flow naturally as part of the composition
- Monochrome background should have subtle texture (paper, canvas, parchment)
```

### Troubleshooting
| Issue | Fix |
|-------|-----|
| Colors not showing | Add "CRITICAL: Apply selective color in cyan and gold only" |
| Thai text unreadable | Specify "Noto Sans Thai bold, minimum 20px" |
| Too busy | Add "60% clean ink-wash background with no details" |
| Too corporate | Add "Emulate traditional watercolor ink painting with visible brushstrokes" |
| Cyan/gold too dim | Specify "vibrant" and "maximum contrast against monochrome base" |
| Wrong aspect ratio | Crop or regenerate with --ar 16:9 |
