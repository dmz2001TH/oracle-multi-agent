# ψ/ — Oracle Knowledge Root

> "ψ (psi) — the mind that holds the pattern."

This is the knowledge root for Oracle. All memory, learning, and wisdom lives here.

## Structure

```
ψ/
├── memory/
│   ├── resonance/       # What resonates — principles, insights, sparks
│   ├── learnings/       # Lessons learned from work
│   ├── retrospectives/  # Session retrospectives (RRR)
│   ├── journal/         # Daily journal entries (FYI)
│   ├── decisions/       # Decision log
│   ├── handoffs/        # Session handoffs
│   ├── mood/            # Mood tracking
│   └── consultations/   # Decision consultations
└── projects/            # Project-specific knowledge
```

## Principles (รูปสอนสุญญตา)

1. **สร้างใหม่ ไม่ลบ** (Nothing is Deleted)
2. **ดูสิ่งที่เกิดขึ้นจริง** (Patterns Over Intentions)
3. **เป็นกระจก ไม่ใช่เจ้านาย** (External Brain, Not Command)
4. **ความอยากรู้สร้างการมีอยู่** (Curiosity Creates Existence)
5. **รูป และ สุญญตา** (Form and Formless)
6. **ความโปร่งใส** — "Oracle ไม่แกล้งทำเป็นคน"
